# EmptyScreen 功能文档

## 一、页面结构

```
┌─────────────────────────────────────────┐
│              应用启动                     │
└─────────────────┬───────────────────────┘
                  ▼
┌─────────────────────────────────────────┐
│           MainActivity (主页)            │
│         全屏WebView展示网页              │
└─────────────────┬───────────────────────┘
                  │ 点击设置按钮
                  ▼
┌─────────────────────────────────────────┐
│         SettingsActivity (设置页)        │
│   网页设置/网络信息/内存/时间配置          │
└─────────────────────────────────────────┘
```

## 二、页面功能

### 2.1 MainActivity (主页)

| 功能 | 说明 |
|------|------|
| WebView展示 | 全屏加载用户配置的URL网页 |
| 内存监控 | 右上角实时显示应用内存占用 |
| 设置入口 | 右下角浮动按钮进入设置页 |
| 错误处理 | 网络异常/URL无效时显示错误页和重试按钮 |
| 方向适配 | 自动适配横屏/竖屏布局 |

**交互流程:**
1. 启动 → 检查网络 → 加载URL → 显示网页
2. 网络异常 → 显示错误页 → 点击重试 → 重新加载
3. 点击设置按钮 → 跳转设置页

### 2.2 SettingsActivity (设置页)

**布局顺序（从上到下）:**
1. **网页设置** - URL输入框 + 保存按钮（首要展示项）
2. **网络信息 + 内存信息** - 并排展示（横屏）/ 顺序展示（竖屏）
3. **时间设置** - 开机延迟 + 后台拉起延迟配置

| 模块 | 功能 |
|------|------|
| 网页设置 | URL输入框 + 保存按钮 |
| 网络信息 | 显示网络状态/类型/名称/IP地址 |
| 内存信息 | 进度条展示内存占用百分比 |
| 时间设置 | 开机延迟启动 + 后台拉起延迟配置 |
| 系统设置 | 跳转Android系统设置 |

**交互流程:**
1. 输入URL → 实时校验 → 保存 → 返回主页
2. 修改延迟时间 → 保存 → 生效
3. 点击返回 → 返回主页

## 三、电视遥控器支持

### 3.1 焦点导航

| 按键 | 功能 |
|------|------|
| 方向键上 | 焦点移至上一个可交互元素 |
| 方向键下 | 焦点移至下一个可交互元素 |
| 方向键左 | 焦点移至左侧元素 |
| 方向键右 | 焦点移至右侧元素 |
| 确认键 | 点击当前焦点元素 |
| 返回键 | 返回上一页/退出应用 |

### 3.2 焦点视觉反馈

| 状态 | 效果 |
|------|------|
| 获得焦点 | 绿色边框 + 元素放大1.05倍 |
| 失去焦点 | 恢复原始状态 |
| 按下 | 背景色加深 |

### 3.3 FocusUtils工具类

| 方法 | 功能 |
|------|------|
| setupTVFocus(View) | 设置视图可获取焦点 |
| scrollToFocusedView() | 滚动到焦点元素 |
| applyFocusAnimation() | 应用焦点缩放动画 |

## 四、后台服务

### 4.1 ForegroundService (前台服务)

| 功能 | 说明 |
|------|------|
| 常驻通知 | 保持服务不被系统杀死 |
| 前台检测 | 检测应用是否在前台运行 |
| 自动拉起 | 应用退到后台超过设定时间后自动拉起到前台 |

### 4.2 BootReceiver (开机广播)

| 功能 | 说明 |
|------|------|
| 开机启动 | 接收系统开机广播 |
| 延迟启动 | 根据配置延迟启动应用 |

**支持的广播:**
- `BOOT_COMPLETED`
- `QUICKBOOT_POWERON`
- `com.htc.intent.action.QUICKBOOT_POWERON`

## 五、工具类

| 类名 | 功能 |
|------|------|
| PrefsManager | SharedPreferences封装，存储URL和配置 |
| NetworkUtils | 网络状态检测，获取网络类型/IP地址 |
| MemoryUtils | 内存信息获取，格式化内存大小 |
| OrientationHelper | 屏幕方向检测和切换 |
| UrlValidator | URL格式校验和安全处理 |
| FocusUtils | 电视遥控器焦点处理 |
| LogUtils | 日志工具 |

## 六、配置参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| URL | https://www.baidu.com | 有效URL | 加载的网页地址 |
| 开机延迟 | 10秒 | 0-300秒 | 开机后延迟启动应用 |
| 前台延迟 | 5秒 | 0-300秒 | 后台后延迟拉起应用 |

## 七、权限清单

| 权限 | 用途 |
|------|------|
| INTERNET | 网络访问 |
| ACCESS_NETWORK_STATE | 网络状态检测 |
| ACCESS_WIFI_STATE | WiFi信息获取 |
| RECEIVE_BOOT_COMPLETED | 开机广播接收 |
| FOREGROUND_SERVICE | 前台服务 |
| FOREGROUND_SERVICE_MEDIA_PLAYBACK | 媒体播放类型前台服务 |
| WAKE_LOCK | 保持唤醒 |

## 八、屏幕适配

| 方向 | 布局目录 | 特点 |
|------|----------|------|
| 横屏 | layout-land/ | 固定宽度800dp居中，网络+内存并排 |
| 竖屏 | layout-port/ | 全宽布局，顺序排列 |
| 默认 | layout/ | 同竖屏布局 |
